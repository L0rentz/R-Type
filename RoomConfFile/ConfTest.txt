# pour ajouter un mob il faut
#
#   path: le chemin jusqu'a la lib
#   type: le type de l'objet
#   time: le timer avant l'apparition du nombre de mob
#   numberToSpawn: le nombre de mob à spawn à l'instant time
#
#   Path    type    time    numberToSpawn
#
# exemple:
# ./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 3 1
./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 3 10
./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 30 10
./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 90 10
./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 120 10
./src/server/entities/BidosSlaves/BidosSlaves.so BidosSlaves 2000 1
#
